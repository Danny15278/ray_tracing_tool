
# Image Ray-Tracing Tool

A ray-tracing image tool developed in modern C++. Based on concepts introduced in Peter Shirley, Trevor David Black, and Steve Hollasch's "Ray Tracing in One Weekend".
After compiling main.cpp, ray-traced image files are produced in the .ppm file format. They can be viewed using a tool such as ImageMagick, Gimp, etc. 

Features so far:
- Sky gradient
- Sphere object
- Scenes with multiple objects
- Surface normalisation
- Pixel antialiasing
